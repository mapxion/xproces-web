Formatos de nube de puntos
LASNube de puntos estándar muy usada en inspección, topografía, CAD y GIS.
LAZLa misma nube que LAS, pero comprimida para ocupar menos espacio.
E57Formato profesional de nube de puntos para compatibilidad con software de ingeniería.
PLYFormato técnico 3D muy usado en software especializado.
PTSFormato sencillo de nube de puntos usado en algunos flujos técnicos.
DXF nubeFormato CAD para ciertos flujos de nube de puntos.
CL3Formato especializado de nube de puntos usado en algunos entornos profesionales.
U3DFormato 3D pensado sobre todo para algunos flujos de documentación.
PDF nubeVista 3D de nube de puntos dentro de un PDF, cuando el flujo lo permita.
OBJ nubeExportación de nube de puntos en OBJ. Menos habitual, pero disponible en algunos flujos.
Modelos 3D y mallas
Malla OBJModelo 3D de superficie con textura, muy compatible con programas 3D.
FBXFormato muy usado en visualización, animación y programas 3D.
STLFormato clásico de malla, conocido por impresión 3D e ingeniería.
DAEFormato Collada para intercambio entre distintos programas 3D.
3DSFormato 3D clásico, útil sobre todo por compatibilidad antigua.
WRLFormato 3D antiguo pero todavía utilizable en algunos flujos concretos.
PDF 3DModelo 3D embebido en PDF para compartir de forma sencilla.
DXF mallaSalida de modelo 3D pensada para ciertos entornos CAD.
Malla PLYOtra forma de exportar la superficie 3D para intercambio técnico.
Malla U3DModelo 3D pensado para algunos flujos de documentación en PDF.
Visualización web
Cesium 3D TilesSalida válida para visualización web 3D jerárquica. Útil cuando el destino final es un visor web.
Ortofotos, elevación y documentación
Ortomosaico TIFFImagen corregida geométricamente, útil para una vista cenital precisa.
Ortomosaico JPGVersión más ligera del ortomosaico, buena para consulta rápida.
DEM/DSM TIFFModelo de elevación más útil en trabajos topográficos y cartográficos.
